# Saurabh Anand Portfolio

A focused React and Vite portfolio site for Saurabh Anand. The site opens directly to the About experience and includes section navigation for Home, Journey, Philosophy, Skills, Beyond Code, Connect, and Resume.

## Tech Stack

- React 18
- Vite
- React Router
- Tailwind CSS
- Lucide React icons

## Getting Started

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

The resume PDF is served from `public/Saurabh_anand_resume.pdf`.
